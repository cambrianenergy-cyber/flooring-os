# My TypeScript Web Application

This is a basic "Hello World" web application built with TypeScript. It serves as a starting point for developing web applications using TypeScript.

## Project Structure

```
my-ts-web-app
├── src
│   ├── index.ts          # Entry point of the application
│   └── types
│       └── index.ts      # Type definitions for the application
├── public
│   └── index.html        # Main HTML file served to the client
├── package.json          # npm configuration file
├── tsconfig.json         # TypeScript configuration file
└── README.md             # Project documentation
```

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd my-ts-web-app
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the application:**
   ```bash
   npm start
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to see the application in action.

## Features

- Type safety with TypeScript
- Basic server setup
- Simple "Hello World" response
